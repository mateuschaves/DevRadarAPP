# Installation

```bash
$ npm install
```

## Technologies

:white_check_mark: React Native\
:white_check_mark: react-native-maps\
:white_check_mark: styled-components\
:white_check_mark: react-native-webview

## Running the app

```bash
# android
$ react-native run-android

# ios
$ react-native run-ios
```
